# VIM模式[6种基本模式+5种派生模式]

## 基本模式
### 普通模式
#### 删除命令[vim中的删除只是将内容剪切到一个缓存区]
1. 删除单一字符 -- x
2. 删除更多多字符 -- d motion
```
motion:
 ^或数字0 -- 将光标定位到行首
 $ -- 将光标定位到行末
 b -- 将光标定位到所在单词起始处[连按会跳到上一个单词起始处]
 e -- 将光标定位到所在单词末尾[连按会跳到下一个单词末尾]
 w -- 将光标定位到所在单词下一个单词起始处
 G -- 将光标定位到文件末尾 
 ```
3. 整行删除 -- dd

#### 撤销
- u -- 撤销最后一次修改
- U -- 撤销对整行的修改
Ctrl+r -- 反撤销
#### 粘贴
- p -- 将最后一次删除的东西粘贴到光标之后
- P -- 将最后一次删除的东西粘贴到光标之前
#### 拷贝-- y [num] motion
- 复制当前行 -- yy
- 从当前行复制到文件末尾 -- yG
- 从当前行复制到文件开头 -- ygg
#### 替换
- 单字符替换 - r
- R--替换模式
#### 修改 - c [num] motion ==> 修改 = 删除【光标定位不一样】+插入模式 
#### 文件信息： Ctrl + g
#### 停止终端输出流：Ctrl + s[按了之后所有的一切按键都傻逼掉了]
#### 恢复终端输出流：Ctrl + q
#### 快速配对括号或大括号等:%
#### 缩进：[num] >> 或 <<
#### 搜索：/search_char  ==> N 上一个;n 下一个
#### 替换：全文见替换==>%s/old/new/g
### 插入模式
> - i -- 在光标前进入插入模式
> - I -- 在光标行首进入插入模式
> - a -- 在光标后边进入插入模式
> - A -- 在光标行末进入插入模式
> - o -- 在光效下方另起一行进入插入模式
> - O -- 在光标上方另起一行进入插入模式
> - s -- 删除光标所在字符进入插入模式
> - S -- 删除光标所在行进入插入模式
