# 揭秘 PC 微信多开

## 技术眼光微信为什么不能开多个？

> https://www.bilibili.com/read/cv6259988/

手机端多开微信相信很多人都知道，像华为、小米等手机系统都对此做了支持；像苹果虽然手机系统本身不支持，但是网上也有一些收费软件可以达到目的（购买需谨慎，存在封号嫌疑）。

如何在运行 `Windows` 系统的电脑上启动多个微信，网上也有很多人给了很多方法，今天来简单说说它们的原理，并且介绍一个更加可靠的方式给大家。


## 微信 `PC` 端为什么不能双开？

其实不仅仅微信，像百度云盘、企业微信等等这些，正常情况下一台电脑上都是只能开启唯一一个这样的程序！从技术的角度来讲这种**确保全局只能有一个程序被启动**的检查机制叫“**单例模式**”。

正常情况下，我们直接手动双击微信图标启动，程序在启动的时候会进行全局单例模式检查，如果发现已经存在微信标识进程，就会直接把对应进程的微信窗口激活，定位到桌面最前面，随后自己退出。

利用 `Process Explorer` 进程管理增强工具定位找到 `_WeChat_App_Instance_Identity_Mutex_Name` 这个句柄，如下图：

![微信进程实例标识](B8B4463B3CC94EDFA2F6C8A95C13A4E9)

发现，只要这个句柄存在那么微信就无法多开。将其关闭后，则可以打开多个

![微信多开](9C1AC6178F0A46A2A709BE33C6EAC999)

> https://www.sohu.com/a/413101567_827544

因此解决该问题的方法就是：在程序启动进行检查时，进行欺瞒！
大致分为三类
1. 拼手速：趁`_WeChat_App_Instance_Identity_Mutex_Name`还没创建的时候，启动多个程序，钻程序单例检查这个短暂间隙的时间空子
2. 东施效颦：抄袭一个一模一样的程序出来，但是对检查逻辑进行修改
3. 掩耳盗铃：通过其他方式将标识进程隐藏起来或杀死，让程序自认为整个电脑只有自己一个进程在运行


## 如何开启多个微信？

### 拼手速
#### 方法一：无需任何工具
1. 首先确保没有登录任何微信，如果已有登录的微信，先彻底退出程序。操作之前务必保证没有正在运行的微信，这是关键点。
2. 点击桌面任意空白处，使微信图标不被选中。
3. 此时按住键盘上的回车键不放，同时鼠标左键点击微信图标，即可同时打开很多个登录窗口，觉得数量够了就放开回车键。点击图标时会有闪烁，若视觉变化不太明显，按住2秒左右也可以打开好几个了。
4. 桌面必须要有微信的快捷方式

![image](564AAA66170A42889AE2994FF925E03A)

优点：
1. 无需任何成本，操作简单

缺点：
1. 打开微信的数量每次是不可控的;
2. 如果使用期间一不小心退出了其中一个微信，再想多开必须将其他微信一并退出重新


#### 方法二：使用脚本快速开启

将下方代码复制，将 `C:\Program Files (x86)\Tencent\WeChat\WeChat.exe` 改成你本机微信的安装地址，保存为 `微信多开.bat` 文件，双击打开即可！

```bash
TASKKILL /F /IM wechat.exe
start "" "C:\Program Files (x86)\Tencent\WeChat\WeChat.exe"
start "" "C:\Program Files (x86)\Tencent\WeChat\WeChat.exe"
start "" "C:\Program Files (x86)\Tencent\WeChat\WeChat.exe"
start "" "C:\Program Files (x86)\Tencent\WeChat\WeChat.exe"
start "" "C:\Program Files (x86)\Tencent\WeChat\WeChat.exe"
```

![image](51014388376049FBBD4FCD69863F70BC)

优点：
1. 能使用到最新版本微信客户端

缺点异常明显：
1. 只适合程序员这么干，因为要修改的时候很扯蛋
2. 要是电脑卡顿了一下，说实话这个概率也是拼手速

### 东施效颦
#### 方法三：下载两个不同的微信客户端

1. 微信官方自己提供的客户端，官方下载地址：https://pc.weixin.qq.com/
2. 去 `Windows` 的 `Microsoft Store` 下载一个 `微信 for Windows` 客户端

![image](B946D7E6E7074466BBDCE6612EA74CB7)

优点：
1. 如果仅仅是聊天，那么确实是够用的
2. 获取方式是最简单的

缺点：
1. `Windows` 的 `Microsoft Store` 不靠谱，很少人使用
2. 由于原因一，导致这个软件就像后妈养的孩子，自 `2016/8/22` 更新之后，再没有更新过，现在算是彻底停止发育，新版客户端的功能它都没有
3. 如果自己开发一个，成本极大
4. 最多只能多打开一个微信，如果你有 3 个微信还是不能解决你的需求



### 掩耳盗铃

#### 方法四：使用程序多开或插件

优点：
1. 随时随地的开启多个应用，不受限制

缺点：
1. 你需要知道“安装路径”这个概念，对普通人有一定的成本
2. 只适用于 Windows 系统
3. 在使用多开打开的程序期间，多开程序不能关闭

**软件1：RevokeMsgPatcher**

![RevokeMsgPatcher](58B9CF199193427D99B1B465CB7AF9DF)

优点：
1. 除了可以多开，还能返撤回（即：好友信息撤回之后你依然可以看到对方的信息）
2. 软件是绿色版本，解压即可使用，无需安装
3. 软件包小巧，仅仅 41K+
4. 代码在 Github（一个公共的代码仓库）上，有作者持续维护着，保障大。短期内不用担心那天突然就不能用了

缺点：
1. 使用本程序前，需要先关闭 微信/QQ/TIM。那么，意味着一不小心退出小号，想重开必须把所有号退了才能重新打开
2. 微信/QQ/TIM 每次更新完之后，需要重新安装补丁。那么意味着补丁在不同版本间不是通用的，那么出错率就较大

> 点击"防撤回"功能：界面可能会无响应，需耐心等待且由于该功能实际是修改了微信的 `WeChatWin.dll` 文件、QQ/TIM 的 `IM.dll` 文件，因此杀毒软件可能会弹出警告，对非互联网人不友好。


**软件2：V5 程序多开器**

![V5 程序多开器](C3C727840C8D4F6BB69A49CBE6EEB5A9)

使用方式：点击【...】选择要打开的程序或**直接拖拽快捷方式到窗口**也可以，然后点击“隔离打开程序”，就能轻易的打开多个程序。

优点：
1. 随时随地的开启多个应用，不受限制
2. 相比`RevokeMsgPatcher`，不依赖任何补丁
3. 能随意打开多个任何独立的软件，而不仅仅只是多开鹅厂系列的产品

缺点：
1. 在使用多开打开的程序期间，多开程序不能关闭
2. 相比`RevokeMsgPatcher`，没有防撤回功能
3. 只适用于 Windows 系统
