## Sping 注解

### @RequestParam这个注解的作用

`Param` 类如下：

```java
@Data
@AllArgsConstructor
public class PackageDetailParam {
    private final String clid;
    private final String id;
    private final String bid;
    private final String lang;
}
```

通过 `@AllArgsConstructor` 注解隐式的生成了全属性构造函数，而 `Controller` 类中的如下：

```java
@RestController
@Controller
@RequestMapping("/checkout/")
@Api(tags = "下单页", produces = "application/json;charset=utf-8")
@AllArgsConstructor
public class MyController {

    private final CheckoutService checkoutService;
    
    @RequestMapping(value = "details", method = RequestMethod.GET)
    @ApiOperation(value = "套餐详细信息")
    @ResponseBody
    public ResultObject<PackageDetailVo> details(
        @ApiParam(value = "搜索id", example = "e3a72a25xxxxxxb", required = true)
        @RequestParam(value = "clid") String clid,
        @ApiParam(value = "报价id", example = "4xx81", required = true)
        @RequestParam(value = "id") String id,
        @ApiParam(value = "mongodb object id", example = "64xxxx38d", required = true)
        @RequestParam(value = "bid") String bid,
        @ApiParam(value = "客户端语言", example = "en", required = false)
        @RequestParam(value = "lang", defaultValue = "en") String lang
    ) throws InterruptedException {
        return ResultUtils.createSuccess(checkoutService.getPackageDetailInfo(new PackageDetailParam(clid, id, bid, lang)));
    }
}
```

有什么注解或其它方式在调用 `new PackageDetailParam` 的时候可以将 `new PackageDetailParam(clid, id, bid, lang)`  替换掉，不需要手动将所有接收到的参数手动写出来传递给构造方法吗？

根据 `ChatGPT` 的提示，通过 `@ModelAttribute` 注解可以解决，通过 `@ModelAttribute` 注解解决后的代码如下：

```java
@RequestMapping(value = "details", method = RequestMethod.GET)
@ApiOperation(value = "套餐详细信息")
@ResponseBody
public ResultObject<PackageDetailVo> details(
    @ApiParam(value = "搜索id", example = "e3a72a2548b7490ca9eb7b2ff597a8eb", required = true)
    @RequestParam(value = "clid") String clid,
    @ApiParam(value = "报价id", example = "422330681", required = true)
    @RequestParam(value = "id") String id,
    @ApiParam(value = "mongodb object id", example = "64489557cffbb95f6955a38d", required = true)
    @RequestParam(value = "bid") String bid,
    @ApiParam(value = "客户端语言", example = "en", required = false)
    @RequestParam(value = "lang", defaultValue = "en") String lang,
    @ModelAttribute PackageDetailParam packageDetailParam
) throws InterruptedException {
    return ResultUtils.createSuccess(checkoutService.getPackageDetailInfo(packageDetailParam));
}
```

## JSON 扩展：Jackson 包

`FastJson` 和 `Jackson` 是两款常用的 `Java` 解析 `JSON` 的工具。

- `Fastjson` 的优势在于解析速度快，解析效率高，可以轻松处理大量的 `JSON` 数据
-  `Jackson` 则更加全面，支持多种数据格式，拥有丰富的序列化和反序列化特性，例如格式化输出、自定义序列化和反序列化等
