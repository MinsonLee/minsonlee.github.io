# 位运算
现代计算机中所有的数据都是通过二进制的形式存储在设备中，位运算就是直接对整数在内存中的二进制位进行操作。

## 二进制
1. 为什么现代电子计算机技术全部采用二进制进行表示数据或数据运算？
> 因为只有 0、1 两个数字符号，非常简单、方便、稳定，只需要利用电波的高低或开关的闭合两种情况即可表示！

## 原码/反码/补码
1. 对于有符号数而言：二进制的最高位是符号位：0表示正数，1表示负数
2. ==0和正数的原码、反码、补码都是一样==。即：正数三码合一
3. 负数的反码=负数原码符号位不变，其它位取反
4. 负数的补码=负数反码 + 1
> - 0 原码==> 0000 0000；反码 ==> 0000 0000；补码 ==> 0000 0000
> - 1 原码==> 0000 0001；反码 ==> 0000 0001；补码 ==> 0000 0001
> - -1 原码 ==> 1000 0001；反码 ==> 1111 1110；补码 ==> 1111 1111
5. ==在计算机运算时，都是以补码的方式来进行运算的==
> `1+1` `1-1`=`1 + (-1)`

## 位运算符
`Golang` 中有 3 个位运算："按位与 `&`"、"按位或 `|`"、"按位异或 `^`"
- **按位与 `&`**： 两者全为 1，结果为 1，否则为 0
- **按位或 `|`**：两者有一个为 1，结果为 1，否则为 0
- **按位异或 `^`**：两者一个为 1，一个为 0，结果为 1，否则为 0

## 位移运算符
位右移 `>>` 和 位左移 `<<`，位移运算规则如下：
- 位右移 `>>`：低位溢出，符号位不变，并用符号位补溢出的高位
- 位左移 `<<`：符号位不变，低位补0

var a int = 1 >> 2 // 0000 0001 ==> 0000 0000 = 0

var a int = 1 << 2 // 0000 0001 ==> 0000 0100 = 4
