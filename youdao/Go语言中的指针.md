字节（Byte）是计算机信息技术用于计量存储容量的一种计量单位，也表示一些计算机编程语言中的数据类型和语言字符。一个字节存储 8 位无符号数(比特位)，存储的数值范围是 0-255。

**计算机中所有的数据都必须放在内存中，不同类型的数据占用的字节数不一样**（PS：int 占用 4 个字节，char 占用 1 个字节；**注意：字节本身是不分数据类型的，但不同的开发语言对不同的数据类型进行了规定其占用的字节数**）。

为了正确地访问这些数据，必须为每个字节都编上号码，就像门牌号、身份证号一样，每个字节的编号是唯一的，根据编号可以准确地找到某个字节。

我们将内存中字节的编号称为**地址**或**指针**。

数据在内存中的地址也称为**指针**，如果一个变量存储了一份数据的指针，我们就称它为**指针变量**。

变量名实际只是为了让人类更加简单、直观的对数据有一定的区分，在编译的时候就会被替换为对应的内存地址。

相较于 C 语言中指针变量定义与读取，Go 语言中的指针变量只有在定义时略有不同。**Go 语言中指针不能运算。**

```C
// C 语言定义指针变量
Type *name;
Type *name = value;

// C 语言中获取一个变量的指针
int a = 100;
int *p_a = &a;

// C 语言中通过指针变量获取数据
*p_a; // 获取指针变量 p_a 的值-100。即：*pointer;
```

```Go
// Go 语言定义指针变量
*Type name
name = &value
// 或者 name := &value

// Go 语言中获取一个变量的指针
var a int = 100
var pA *int
pA = &a

// Go 语言中通过指针变量获取数据
*pA; // 获取指针变量 pA 的值-100。即：*pointer;
```

**Go 语言中的参数传递只有值传递一种方式。**
