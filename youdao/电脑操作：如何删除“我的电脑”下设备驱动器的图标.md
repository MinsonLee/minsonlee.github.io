1. `Win + R` 打开`运行`，输入`regedit`打开`注册表编辑器`
2. 找到`HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\MyComputer\NameSpace\`，删除对应目录即可

## 清理 DNS


### Linux 下清空 DNS 缓存

> Linux 下 DNS缓存实现通常有两种方式：① DNS缓存程序NSCD(name service cache daemon)；② 
用 Bind/dnsmasq 架设 Caching Name Server 实现。

#### 清除 NSCD 上的缓存

```sh
service nscd restart
# 或 /etc/init.d/nscd restart
```

#### 清除 BIND 服务器上的缓存

```sh
rndc flush
```

#### 清除 DNSmasq 缓存

```sh
sudo /etc/init.d/dnsmasq restart
```
> DNSmasq 是一个轻巧、易用的 DNS 服务工具，它可以应用在内部网和 Internet 连接的时候的 IP 地址 NAT 转换，也可以用做小型网络的 DNS 服务


### Windowns 下清理 DNS 缓存

```sh
# 清理缓存
ipconfig /flushdns

# 查看 DNS 缓存内容
ipconfig /displaydns
```
> Windows 下的 DNS 服务是由 DNS Client 进程控制的，可以在服务中进行重启/关闭(每次连接 Internet 都会进行 DNS 查询) ==> 运行 `Win+R` > 输入 `services.msc` > 找到 `DNS Client`
    ![image](4B558AB6BAD74687BA70CFCA6CD667C2)


### Mac 下清理 DNS 缓存

```sh
# Tiger或更低版本 Mac OS
sudo lookupd -flushcache

# Leopard和Snow Leopard
sudo dscacheutil -flushcache 

# Lion、Mountain Lion和Mavericks
sudo killall -HUP mDNSResponder 

# 
sudo discoveryutil mdnsflushcache  
```

## Win/E 快捷键替换

win+e 替换为 xyplorer 为例。


1.  首先命令行或运行(win+R)输入 `regedit` 打开注册列表

2.  在注册表中找到 `计算机\HKEY_CLASSES_ROOT\Folder\shell\opennewwindow\command`

3.  将 `(默认)` 项的值改为 `xyplorer 安装路径`，例如修改为 `D:\apps\XYplorer\XYplorer.exe %2`。**注意：后边的 ` %2` 是参数，切勿缺少！！！**
    ![image](08894E4356A14300A340A078CC9D2C72)

4. 删除 `DelegateExecute` 项

**注意：** 部分系统重启后还是会自动生成 `DelegateExecute` 项，导致 xyplorer 失效，可以将 `DelegateExecute` 的值修改为 `任意不存在的值`，如：`1`。 

系统自身的资源管理器 `%SystemRoot%\explorer.exe "%1"`
