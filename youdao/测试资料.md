### 拦截HTTPS注意事项
1. 手机访问 [chls.pro/ssl](chls.pro/ssl) 的时候要使用IPhone自带浏览器`Safari`,如果使用自行安装的Chrome浏览器可能会导致：浏览器下载证书之后无法唤起系统设置进行证书安装

2. 默认情况下Charles是不开启HTTPS协议拦截的,如果需要对抓取的网站进行HTTPS请求进行抓取，还需要：``右击拦截的链接` 》 `Enable SSL Proxying`

3. iPhone 安装证书后需要设置信任证书才能正常抓取 HTTPS 
> - IOS 10.3 之前：`设置` >> `通用 >> 描述文件与设备管理 >> 点击证书 "信任"
> - IOS 10.3之后还需要额外设置：`设置` >> `通用` >> `关于本机` >> `证书信任设置` >> `打开Charles证书信任开关`

4. 若不抓包时请关闭手机 HTTP 代理，否则断开与电脑连接后会造成无法上网的情况
