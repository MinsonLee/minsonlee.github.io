## 常用的进制
对于**整数**，计算机中常见的进制有四种表达方式：二进制、八进制、十进制、十六进制

### 二进制
- 由 `0`,`1` 两个数字组成
- 满 2 进 1
- 在 golang 中，不能直接使用二进制表示一个整数，沿用了 C 的特点
    > ==为什么呢？==

### 八进制
- 由 `0-7` 八个数字组成
- 满 8 进 1
- 以数字0开始。如：`011` 默认表示八进制中的 `9`

### 十进制
- 由 `0-9` 十个数字组成
- 满 10 进 1

### 十六进制
- 由 `0-9` 和 `A-F`，十个数字及六个大写(或小写)字母表示
- 满 16 进 1
- 以 `0x` 或 `0X` 开头表示。如：`0XA` 表示十进制中的 10;`0X11` 表示十进制中的 17
- `A-F` 六个字母不区分大小写


## 常用进制转换
### 其他进制转十进制：加权求和
**规则：从==最低位开始(最右边)==，将每个位上的数字`X`提取出来，乘以 `N` 的 `C` 次方，然后==求和==。**
> - `N` 表示进制，如：二进制则 `N` 表示 `2`；
> - `C` 等于 `(位数 - 1)`

1. 二进制 1011 转成十进制整数
```math
1011 = 1*2^0 + 1*2^1 + 0*2^2 + 1*2^3 = 1 + 2 + 0 + 8 = 11
```

2. 八进制 0123 转成十进制整数
```math
0123 = 3*8^0 + 2*8^1 + 1*8^2 = 3 + 16 + 64 = 83
```

3. 十六进制 0X34AF 转成十进制整数
```math
0X34AF = 15*16^0 + 10*16^1 + 4*16^2 + 3*16^3 = 15 + 160 + 1024 + 12288 = 13487
```

### 十进制转其他进制：短除法
短除法：将该数不断除以`N`，==直到商为0为止==，然后将每步得到的==余数倒过来==，就是对应 `N进制` 的数

1. 将 56 转成二进制 `11000`
```math
56 \div 2 = 28 ...... 0

28 \div 2 = 14 ...... 0

14 \div 2 = 7  ...... 0

7  \div 2 = 3  ...... 1

1  \div 2 = 0  ...... 1
```

2. 将 156 转成八进制 `0234`
```math
156 \div 8 = 19 ...... 4

19  \div 8 = 2  ...... 3

2   \div 8 = 0  ...... 2
```

3. 将 356 转成十六进制 `0x164`
```math
356 \div 16 = 22 ...... 4

22  \div 16 = 1  ...... 6

1   \div 16 = 0  ...... 1
```

### 二进制转八/十六进制
**规则：将`二进制`数每`X位一组`（低位开始组合，不足使用0补足高位），转成对应的`N进制`数即可。**
> 如，将：二进制转为八进制，即每3位一组(2^3=8)；二进制转为十六进制，即每4位一组

1. 将二进制数 `11010101` 转为八进制
```math
11\ \ 010\ \ 101 = 011\ 010\ \ 101 = 0\ \ 3\ \ 2\ \ 5
```

2. 将二进制数 `11010101` 转为十六进制
```math
1101\ \ 0101 = 0X\ \ D\ \ 5
```


### 八/十六进制转二进制
**规则：将对应进制数每1位转成对应的 X 位二进制数，不足补0即可**
> 如，将：八进制转为二进制，即每1位转为3位二进制数；十六进制转为二进制，即每1位都转为4位二进制数

1. 将 0237 转为二进制数
```math
0237 = 010\ \ 011\ \ 111 = 10 011 111
```

2. 将 0x237 转为二进制数
```math
0x237 = 0010\ \ 0011\ \ 0111 = 10 0011 011
```
